# MusicBox - YouTube Music Player

Музыкальный плеер на базе YouTube с локальным хранением плейлистов и лайков.

## Возможности

- 🔍 **Поиск треков** - поиск музыки на YouTube
- ❤️ **Лайки** - сохранение любимых треков локально
- 📋 **Плейлисты** - создание и управление плейлистами
- 🎵 **Воспроизведение** - проигрывание аудио из YouTube
- 📥 **Импорт из CSV** - импорт треков из CSV файла

## Технологии

- WPF (.NET 8)
- YoutubeExplode - для работы с YouTube
- NAudio - для воспроизведения аудио
- MaterialDesignThemes - для UI

## Использование

1. Введите название трека в поиск
2. Выберите трек из списка для воспроизведения
3. ПКМ на треке → добавить в лайки или плейлист
4. Создайте плейлист через кнопку "Создать плейлист"

## Импорт из CSV

Формат файла:
```
Название,Исполнитель
Song Title,Artist Name
Another Song,Another Artist
```

## Хранение данных

Все данные (лайки, плейлисты) хранятся локально в:
`%LocalAppData%\MusicBox\library.json`
