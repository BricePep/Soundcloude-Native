FAQ

Частые вопросы

Ответы на то что спрашивают чаще всего

Нужно ли устанавливать .NET отдельно?
Нет. Приложение self-contained — все зависимости встроены в установщик. Просто скачай и запусти.
Работает ли Mica на Windows 10?
Mica — фича Windows 11. На Windows 10 приложение работает с обычным тёмным фоном. Все остальные фичи доступны на обеих ОС.
Это официально от SoundCloud?
Нет, SC Native — неофициальный клиент. Не аффилирован с SoundCloud Ltd. Использует публичное API для авторизации и стриминга.
Мои данные в безопасности?
Все настройки хранятся локально на твоём компьютере. Никакие данные не отправляются на сторонние серверы (кроме баг-репортов и предложек по твоей инициативе).
Можно ли слушать музыку без аккаунта?
Для стриминга с SoundCloud нужен аккаунт. Но локальные файлы можно добавлять и без авторизации.
Как предложить идею или сообщить о баге?
Прямо из приложения — кнопки «Сообщить» и «Предложить» в сайдбаре. Отправляется мгновенно в наш Discord.
